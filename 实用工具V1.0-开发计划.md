> **基于 PRD 版本**：V1.0 | **编写日期**：2026-06-29 | **开发负责人**：Senior Developer

## 一、技术选型

| 层面 | 选择 | 理由 |
|------|------|------|
| **框架** | React 18 + TypeScript | 社区生态丰富，工具函数库完备（lodash、date-fns、big.js 等），适合组件化工具页面 |
| **构建工具** | Vite 5 | 秒级 HMR，原生 ESM，开发体验最佳 |
| **路由** | React Router v6 | 支持嵌套路由、懒加载，适合「首页→分类→工具详情」三级页面结构 |
| **状态管理** | Zustand | 轻量无 boilerplate，适合管理搜索状态、主题、收藏/历史等简单全局状态 |
| **CSS 方案** | Tailwind CSS 3 + CSS Variables | 原子化开发效率高，CSS Variables 支撑暗黑模式无缝切换 |
| **UI 组件库** | 不引入重型组件库，自建 + Headless UI | 工具平台的 UI 以表单/输入区为核心，重型组件库反而增加体积 |
| **工具函数** | date-fns / big.js / crypto-js / jschardet / pinyin-pro | 分别覆盖日期、精确计算、加密、编码检测、拼音等场景 |
| **代码编辑器** | Monaco Editor（仅 JSON/JS/HTML/SQL 格式化工具用） | 专业代码编辑体验，支持语法高亮、折叠、错误提示 |
| **图片处理** | browser-image-compression + qrcode + jsbarcode + tesseract.js | 纯前端压缩、二维码/条形码生成、OCR 文字识别 |
| **部署** | Vercel / CloudStudio 静态部署 | 纯前端 SPA，无后端依赖，任何静态托管均可发布 |

### 2.2 路由设计

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | 首页 | 分类卡片矩阵 + 搜索 + 热门推荐 |
| `/category/:slug` | 分类列表 | 左侧导航 + 右侧工具卡片 |
| `/tool/:toolId` | 工具详情 | 具体工具操作页面 |
| `/search?q=xxx` | 搜索结果 | 搜索匹配结果列表 |

### 2.3 工具注册表设计（tool-registry.ts）

所有 110+ 工具统一注册，支持搜索、路由、分类：

```ts
interface ToolEntry {
  id: string;           // 唯一标识，如 'text-case-convert'
  name: string;         // 工具名称
  category: Category;   // 所属分类
  description: string;  // 一句话功能说明
  keywords: string[];   // 搜索关键词（含别名）
  component: () => Promise<ToolComponent>;  // 懒加载组件
  isCore: boolean;      // 是否 V1.0 核心工具
  popularity: number;   // 初始热度权重（热门推荐排序）
}
```

## 五、技术风险与应对

| 风险 | 影响 | 应对方案 |
|------|------|----------|
| **OCR 准确率不足** | tesseract.js 对复杂排版中文识别率偏低 | 1. 接入第三方 OCR API（百度/腾讯）作为加强选项；2. 产品文案标注"建议清晰扫描件" |
| **汇率/油价/放假数据实时性** | 第三方 API 不稳定或无免费接口 | 1. 汇率：接入 exchangerate-api 免费层；2. 油价/放假：静态数据 + 定期手动更新 + 标注更新日期；3. 预留 API 降级为静态数据方案 |
| **视频解析合规风险** | 可能涉及侵权 | 仅提供公开分享链接解析，不支持私有/付费内容，页面底部加免责声明 |
| **大文本输入卡顿** | 批量处理 10 万行时页面假死 | Web Worker 将计算移到后台线程，主线程仅渲染结果摘要 |
| **移动端触控精度** | 二维码/条形码下载、颜色取色器等桌面操作在移动端体验差 | 移动端适配：触摸友好的按钮尺寸（≥44px），长按复制替代 hover |
| **暗黑模式颜色还原** | 部分工具（颜色值转换、图像处理）暗黑模式下色彩感知偏差 | 对色彩敏感工具强制保持浅色背景，不受主题影响 |
